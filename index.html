Alfred Annict Workflow

アニメ視聴記録サービスAnnictの作品ページに素早くアクセスするためのAlfredワークフローです。ご利用にはAlfred Powerpackのライセンス購入が必要ですのでご注意ください。

作品検索

AlfredからAnnictのデータベースに登録されている作品を検索できます。

「見てる」作品のページに素早くアクセス

ステータスが「見てる」の作品一覧を表示してページに素早くアクセスできます。

必要なもの

  • Powerpack購入済みのAlfred 2以上
  • Annictアカウント

インストール方法

  1. ダウンロードページより最新版をダウンロードします。

  2. ダウンロードしたAnnict.alfredworkflowをダブルクリックすると設定画面が開きます。

  3. Importボタンをクリックします。

  4. ~/.annict/config.jsonにアクセストークンを記述します。

{
  "access_token": "<ACCESS_TOKEN>"
}

使い方

1. Alfredの検索窓を開きます。

2. annくらいまで入力するとAnnictのアイコンがついた「作品検索」と「見てる作品」が候補に出ると思います

3. 「作品検索」を選択するとannict worksが補完されるので、ここに続けて検索したいキーワードを入力すると候補が表示されます

3. 「見てる作品」を選択するとannict watchingが補完され、ステータスが「見てる」の作品が表示されます

4. 作品を選択すると既定のブラウザが起動しAnnictで作品の個別ページが開きます
