# Alfred Annict Workflow
アニメ視聴記録サービスAnnictの作品ページに素早くアクセスするためのAlfredワークフローです。ご利用にはAlfred Powerpackのライセンス購入が必要ですのでご注意ください。

- [Alfred for Mac](https://www.alfredapp.com)
- [アニメ視聴記録サービスAnnict](https://annict.com/)
